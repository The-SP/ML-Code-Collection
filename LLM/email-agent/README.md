# Email Assistant Agent

AI-powered email management using LangGraph and IMAP.

## Setup

1. Install dependencies:

```bash
uv sync
```

2. Copy and configure environment:

```bash
cp .env.sample .env
# Edit .env with your credentials
```

3. Run:

```bash
uv run main.py
```

## Features

- List unread emails
- Get email content
- Mark emails as read
- Summarize emails with AI
